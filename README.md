# 현대 태양광 시스템
## 프로젝트 설명 
1. 목적 : 각 사용자가 태양광으로 충전한 전기의 소비내역을 표 및 다이어그램으로 확인할 수 있으며, 관리자 페이지로 모델 등록및 사용을 제어할 수 있는 관리자 페이지 제작  
2. 인원 : 퍼블리셔 2명, 백엔드 개발 2명, 디자이너 1명
3. 기간 : 2024.06~2024.09 (3개월)
4. 페이지 규모 : 56페이지
5. 팀내 역할 : 퍼블리싱 및 UI팀 리딩
