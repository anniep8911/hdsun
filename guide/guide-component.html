테이블(리스트)

전체적인 테이블 구조는 다음과 같습니다.

✔ 전체적 테이블을 감싸는 요소는 div태그를 이용, 클래스명은 table입니다.
<div class="table">

✔ 행을 먼저 나누게 되는데, 크게 제목과 내용으로 나뉩니다.
✔ 제목은 div를 사용하며, 클래스 명은 tbl-header입니다. 제목영역에는 전체선택과 버튼들이 같이 포함되어있습니다
 <div class="tbl-header">

✔ 테이블 위의 전체 선택부분과 버튼이 모여있는 부분은 아래와 같이 제목의 형제단위로 작성합니다.
  <div class="sel-all">
    <input type="checkbox" name="select-all" id="sel-all"> <label for="sel-all">전체선택</label>   </div>
  <div class="btns">
    <button type="button" class="table-btn del" target-modal=".modal.del">삭제
    <button type="button" class="table-btn down" target-modal=".modal.confirm">다운로드
    <a href="#" class="table-btn new">신규
  </div>

✔ 테이블타이틀은 ul태그로 작성하며, 앞으로 테이블 '행'의 기준은 ul 태그, '열'의 기준은 li태그로 보시면 됩니다.
  <ul>
    <li><input type="checkbox" name="list-check"></li>
    <li>ID</li>
    <li class="w300">Home명</li>
    <li>상태</li>
    <li>고객명</li>
    <li>지역</li>

✔ 이때 '열'의 기준인 li의 카테고리 및 내용이 [영문18글자, 국문 14글자]를 준수해야한다면 class=w300을 추가합니다.
타이틀을 기준으로 셀의 내용에 대한 li에도 동일하게 추가합니다
    <li class="w300">위치</li>
    <li>등록일</li>

✔ 맨 마지막 버튼 부분이 3개일땐 w300클래스명을, 3개 초과일땐 w350클래스명을 추가합니다
    <li class="w300"></li>
  </ul>
 </div>

✔ 테이블의 데이터 내용은 위에 설명한 타이틀과 동일하며, 클래스는 tlb-list로 구분됩니다.
  <ul class="tbl-list">
    <li><input type="checkbox"></li>
    <li>아이디노출됩니다</li>

✔ !여기서 중요한점은 title li에 사용한 n번째의 li에 똑같이 w300 혹은 w350을 추가합니다!
    <li class="w300">ABCDEFGHIJKLMNOPQR</li>
    <li>상태</li>
    <li>고객명</li>
    <li>국내</li>
    <li class="w300">경기도 오산시 수목원로 606</li>
    <li>2024-11-30</li>
    <li class="w300">
      <a href="#" class="small-btn">상세보기
      <a href="#" class="small-btn">정보변경
      <button type="button" class="small-btn del" target-modal=".modal.del">삭제
    </li>
  </ul>
</div>
